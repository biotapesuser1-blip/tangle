## Identity

The Identity module, akin to ENS (Ethereum Name Service), is designed to link identities to wallets, ensuring blocks can be processed securely.

### How Identity Works
Validators must possess an Identity to execute their blocks. Without it, block execution will fail.

Future versions will expand Identity usage to include non-validator participants, aligning with the rollout of delegation capabilities.
